# Ethiopia Financial Inclusion Forecast

Forecast financial inclusion rates (Access & Usage) for Ethiopia 2025-2027.

## Setup

1. Clone repo
2. Install: `pip install -r requirements.txt`
3. Run dashboard: `streamlit run dashboard/app.py`

## Project Structure

- `data/` - Raw and processed data
- `notebooks/` - Analysis notebooks
- `src/` - Python modules
- `dashboard/` - Streamlit app
- `tests/` - Unit tests

## Tasks

1. Data Exploration & Enrichment
2. EDA
3. Event Impact Modeling
4. Forecasting
5. Dashboard
